Class Rish::Interactive
In: lib/rish/interactive.rb
Parent: Object

Rish‘s own interactive mode.

Similar to IRB, but much smaller. The advantage it has is that it doesn‘t look for line continuations and works better with rish‘s built-in commands (well, is this a feature or a bug?)

Also, if the line ends in ";", the result is not plotted, just as in matlab.

hook=(h)

Add a hook which is called before and after each line.

The hook object as to have a before and after method. The before method is called with the input line and has to return it or a changed version. Likewise, the after method gets the result of the evaluation and has to return the value or a changed version.
