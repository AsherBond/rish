Module Rish::Autoreload
In: lib/rish/autoreload.rb

This module tracks loaded files and their timestamps and allows to reload files which have changed automatically by calling reload.

There is nothing magically happening here. Basically, you can reload a file by removing it from $" (the list of all loaded files) and require‘ing it again.
